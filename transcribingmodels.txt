🛠 Downloading Bigger Models

Open PowerShell in your whisper.cpp folder:

cd D:\Project\VoiceCX\voicecx-insights\basemvp\whisper.cpp\models


Run the download script with the model name you want:

.\download-ggml-model.cmd base.en
.\download-ggml-model.cmd small.en
.\download-ggml-model.cmd medium.en
.\download-ggml-model.cmd large-v2


✅ This will save .bin files in the whisper.cpp folder (same place where ggml-tiny.en.bin is).
